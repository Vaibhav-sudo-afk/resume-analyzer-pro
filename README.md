# AI Resume Analyzer Pro 🚀

A premium, interactive front-end application designed to simulate an AI-powered resume analysis process. Built with state-of-the-art web design principles, this project showcases deep expertise in Vanilla JavaScript, modern CSS architecture, and highly engaging user experiences.

![Resume Analyzer Pro Showcase](https://placehold.co/1200x600/0f111a/10b981.png?text=AI+Resume+Analyzer+Pro)

## ✨ Features

- **Stunning UI/UX**: A highly polished "Glassmorphism" interface built entirely with Vanilla CSS.
- **Dark Mode Aesthetic**: Custom dynamic glowing gradients and a sleek dark theme specifically tailored to appeal to modern tech companies.
- **Interactive Drag & Drop**: Native drag-and-drop file upload functionality with satisfying micro-interactions.
- **Mock Analysis Engine**: A simulated, multi-stage loading sequence that mimics backend AI processing (Extracting text -> Analyzing keywords -> Evaluating ATS -> Generating insights).
- **Dynamic Score Animation**: Custom SVG radial progress bar that fluidly animates the final ATS compatibility score.
- **Actionable Insights**: Neatly categorized display for "Detected Skills", "Key Strengths", and "Areas to Improve".
- **Zero Dependencies**: 100% Vanilla HTML, CSS, and JavaScript. No bulky frameworks or libraries.

## 🛠️ Built With

- **HTML5** - Semantic markup and accessibility.
- **CSS3** - Flexbox, CSS Grid, custom variables, keyframe animations, and glassmorphism.
- **Vanilla JavaScript** - DOM manipulation, Event Listeners (Drag & Drop), and asynchronous simulated workflows (`setTimeout` / `Promises`).
- **Google Fonts** - *Outfit* typeface for a tech-forward aesthetic.
- **FontAwesome** - Vector icons for visual clarity.

## 💻 Running Locally

To run this project locally, no build steps or servers are required!

1. Clone the repository:
   ```bash
   git clone https://github.com/Vaibhav-sudo-afk/resume-analyzer-pro.git
   ```
2. Navigate to the directory:
   ```bash
   cd resume-analyzer-pro
   ```
3. Open `index.html` directly in your favorite web browser.

## 🧠 Why This Project?

This project was built to demonstrate proficiency in core web technologies. By avoiding frameworks like React or Tailwind, it highlights a deep understanding of native web APIs, responsive CSS layouts, and complex DOM state management.
